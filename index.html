What is Git?

Git is a free and open source version control system that helps you track changes in your code and collaborate with others. got it???

Common Git Commands

  • git init - Initialize a new Git repository
  • git clone <url> - Copy a repo from GitHub
  • git status - Show changes in your working directory
  • git add . - Stage all changes
  • git commit -m "message" - Save your changes
  • git push - Upload changes to GitHub
  • git pull - Download changes from GitHub

Why Use Git?

  • Track and revert changes easily
  • Collaborate with others through GitHub
  • Backup your code online
  • Manage different features or versions with branches
